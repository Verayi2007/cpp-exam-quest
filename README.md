# C++ 期末闯关 IDE

本工具把两天冲刺的 18 道核心题做成闯关练习。通过当前关才会解锁下一关。

- 本地打开时，提交代码后由本地服务调用 `g++` 编译并运行测试用例。
- GitHub Pages 公网版会把代码提交到 Judge0 隔离沙箱运行，不会在仓库作者的电脑上执行访客代码。

## 启动

双击 `启动闯关IDE.bat`，然后打开：

```text
http://localhost:4173
```

也可以在当前目录执行：

```text
node server.js
```

## 发布到 GitHub Pages

仓库包含 `.github/workflows/pages.yml`。将代码推送到 GitHub 的 `main` 分支后，在仓库 `Settings > Pages` 中将发布来源设置为 `GitHub Actions`。后续推送会自动更新网页。

公网版无需运行 `server.js`，也无需访客安装 C++ 编译器。
